# Contributing

Thank you for your interest in contributing!

## How to contribute

1. Fork the repository  
2. Clone your fork  
3. Create a new branch  
4. Make your changes  
5. Commit your changes  
6. Push to your fork  
7. Open a Pull Request  

## Rules

- Do NOT upload PDF files (they may contain sensitive data)  
- All test data must be anonymized (no personal information)  

## Optional

- Run tests locally if possible  
- Contact maintainers if you have questions  

Thank you for contributing!
